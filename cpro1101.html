CPRO 1101 | Front End Web Development I

Course Description

Use modern web technologies to create, format, design, and publish webpages.

Topics include ethics, site management, information literacy, website usability, graphic design, web design and development process, HTML and CSS, and JavaScript programming.

Credits: 3 | Prerequisite: None | Co-requisite: None

Learning Outcomes

  • Create web pages using HTML and other tools.
  • Implement Cascading Style Sheets (CSS) to format web pages.
  • Design and publish a website according to client requirements.
  • Evaluate strategies for communicating information effectively using web technologies.
  • Examine different languages and frameworks for constructing websites and mobile applications.
  • Explain the principles of responsive and optimized web design.
