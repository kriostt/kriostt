CPRO 1122 | General Mathematics

Course Description

Explore topics in arithmetic, algebra, linear graphics, measurement, and statistics using a problem-solving approach.

Topics include operations, counting, transformations, probabilities, sets, relalations, functions, number systems, logic, and random variables.

Credits: 3

Course Description

  • Perform fundamental arithmetic operations and estimate answers for reasonableness.
  • Use percentages, ratios, and proportions in various applications.
  • Convert and apply imperial and metric measurements.
  • Collect, organize, and interpret data using tables, graphs, and simple statistical measures.
  • Apply properties of probabilities.
  • Manipulate and evaluate algebraic expressions.
  • Apply equations and formulae to problem solving.
  • Write, read, interpret, and apply procedural definitions of functions.
