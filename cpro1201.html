CPRO 1201 | Python Programming I

Course Description

Design, test, debug programs, control the flow of a program, implement array processing and develop programs that process data from files using Python programming language.

Topics include data types and variables, input and output, functions and structures, boolean expressions, file handling, exceptions, object-oriented programming, ergonomics: seating and workspace/workstation, and organization for time management.

Credits: 3 | Prerequisite: None | Co-requisite: CPRO 1121

Course Learning Outcomes

  • Describe the basic principles of the Python programming language.
  • Design, test, and debug progams using Python.
  • Implement function, modules, and other structural elements in a Python programming environment.
  • Implement file input/output, data management, and exception handling in Python.
  • Implement object-oriented programming concepts.
  • Explore principles of time management, personal organization, and self-care while working at a computer.
