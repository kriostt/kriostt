CPRO 1301 | Database Design and SQL

Course Description

Explore relational database design and Structure Query Language (SQL) to construct models, modify database objects, and develop database applications.

Topics include SQL, relational databases, data retrieval and management, system development life cycle, DDL statements, database objects, databse security, control structures, and documentation.

Credits: 3 | Prerequisite: None | Co-requisite: None

Course Learning Outcomes

  • Explain the role of structured query language and relational databases.
  • Amalyze user requirements and construct data models using entity-relationship diagrams (ERD).
  • Use Data Definition Language (DDL) to create and modify the structure of database objects.
  • Use Data Manipulation Language (DML) to add, delete, and modify data in a database.
  • Prepare technical documentation for a database project.
  • Develop single-table and multiple-table databse queries.
  • Implement views and indexes.
