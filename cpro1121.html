CPRO 1121 | Problem Solving / Program Logic

This class was as enjoyable as I thought it would be. To be clear, that's a good thing! As someone that enjoys solving puzzles, I found that the exercises and assignments in this class were almost like a fun game of sudoku. Although drawing the flowcharts was quite confusing at first, it was still interesting to try and figure out what order of events logically makes sense to solve the problem at hand. I think this class did a great job on introducing me to the skills that are needed to be a computer programmer.
