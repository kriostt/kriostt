CPRO 1121 | Problem Solving / Program Logic

Course Description

This course introduces computer-based concepts of problem-solving, structured programming logic and techniques, algorithm development, and program design.

Topics include program flowcharting, algorithms, input/output techniques, looping, functions, selection structures, file handling, control breaks, pseudo coding, and user documentation.

Credits: 3 | Prerequisite: None | Co-requisite: CPRO 1201

Learning Outcomes

  • Design programs using a top-down modernized approach.
  • Control the flow of a program with decision and repetitive structures.
  • Demonstrate the use of logic and problem-solving principles as they relate to computer programming.
  • Develop programs that process data from files including control break processing.
  • Design and imprement algorithms.
  • Understand functions, argument passing, and return types.
